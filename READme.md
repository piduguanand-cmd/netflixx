# NetflixX 🎬

NetflixX is a Netflix-inspired movie browsing web application developed using HTML, CSS, and JavaScript. The project recreates the look and feel of a modern OTT streaming platform, allowing users to explore movie collections through an attractive and responsive interface.

The website features a user authentication system, Netflix-style homepage, movie search functionality, movie details page, Favorites section, and My List section. Movies are organized across multiple languages including Telugu, Hindi, and English. While the platform does not stream movies, it provides an engaging user experience similar to popular streaming services.

## Features

- User Login and Signup Pages
- Netflix-Inspired User Interface
- Hero Banner Section
- Search Movies Functionality
- Movie Details Page
- Favorites Management
- My List Management
- Multi-Language Movie Collection
- Responsive Design for Mobile and Desktop
- Local Storage Integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage

## Project Objective

The main objective of NetflixX is to demonstrate frontend web development skills by building a professional and interactive movie browsing platform that mimics the design and user experience of Netflix.
